# Project 6.2: SC Decoder N=8 RTL Baseline

## Objective

This project implements a combinational RTL baseline for an N=8 Polar SC decoder and verifies it against golden vectors generated by the Python behavioral model from Project 6.1.

## Files

- RTL: `rtl/sc_decoder_n8.v`
- Golden model: `model/sc_decoder_n8_golden.py`
- Testbench: `tb/tb_sc_decoder_n8_vectors.v`
- Simulation script: `sim/run_sc_decoder_n8_vectors.sh`
- Golden vectors: `tests/golden_vectors/sc_decoder_n8_vectors.csv`
