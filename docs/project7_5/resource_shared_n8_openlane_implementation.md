# Project 7.5: OpenLane Implementation Of Resource-Shared SC Decoder N=8

## 1. Objective

This project implements the resource-shared scheduled SC Decoder N=8 using OpenLane.

The goal is to evaluate whether the resource-shared architecture provides better physical implementation characteristics than the combinational N=8 baseline.
